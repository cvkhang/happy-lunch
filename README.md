# Happy-lunch

Happy-lunch is a web application that helps users discover restaurants and view detailed information about them.

## Features

- View a list of restaurants.
- View detailed information for each restaurant (Menu, Reviews, Location).
- Search and filter restaurants (Upcoming).

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL, Supabase

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation    

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```

2.  **Backend Setup:**
    ```bash
    cd backend
    npm install
    npm start
    ```
    The server will start on `http://localhost:5000`.

3.  **Frontend Setup:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.
